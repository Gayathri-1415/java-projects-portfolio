Student Management System (Java)

A console-based application built using Core Java to manage student records efficiently.

📌 Features
Add Student (with duplicate ID validation)
View all students
Search student by ID
Update student details
Delete student
Filter students by section

🧠 Concepts Used
Object-Oriented Programming (Classes & Objects)
ArrayList (Dynamic data handling)
Iterators (Safe deletion)
Menu-driven programming
Basic validation logic

⚙️ How It Works
Student class → represents individual student data
StudentManager → handles all operations (add, delete, search, update)
Main → manages user input and menu flow

📌 Note
This is a beginner-level project focused on strengthening core Java and problem-solving skills.
